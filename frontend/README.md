# StockIQ Frontend 

This project is a cleaned React + TypeScript + Vite frontend for the StockIQ fintech which i have done as needed by sprint task.

## Pages Included
- Login page
- Dashboard page
- NVDA stock detail page

## What was done
- Built one clean React app instead of separate exports
- Added React Router navigation
- Used simple component structure for readability
- Kept the project UI-only for Sprint 1
- Added responsive layout for mobile view

## How to run locally
```bash
npm install
npm run dev
```

## How to build
```bash
npm run build
```

## Notes
- This is a frontend-only submission.
- No real backend authentication or live stock API is connected.
- Static sample data are used which is used to match sprint requirement safely.
